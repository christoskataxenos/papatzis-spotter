**Papatzis Spotter** is a comprehensive **AI Slop-Prevention Ecosystem** designed to detect and block low-quality, prompt-engineered code. It combines deep AST analysis, statistical heuristics, and global AI agent enforcement to preserve human craftsmanship in the age of LLMs.

Το **Papatzis Spotter** είναι ένα ολοκληρωμένο **οικοσύστημα προστασίας από το AI Slop**, σχεδιασμένο να εντοπίζει και να μπλοκάρει κώδικα χαμηλής ποιότητας. Συνδυάζει βαθιά AST ανάλυση, στατιστικά heuristics και επιβολή κανόνων στους AI agents για τη διατήρηση της ανθρώπινης ποιότητας στην εποχή των LLMs.

## 🎯 Why Papatzis Spotter? / Γιατί το Papatzis Spotter;

### 🇺🇸 The Motivation
In the age of generative AI, codebases are being flooded with "slop" — low-effort, generic code that lacks "soul," character, and deep intentionality. 
- **The "Papatzis" Trap**: AI acts like a digital "papatzis" (con artist), producing code that looks professional and symmetrical on the surface but often lacks the organic irregularities and deep reasoning of human thought.
- **Preserving Craftsmanship**: This tool is built to champion quality over quantity. It's for developers who believe that every line of code should reflect care, detail, and human logic.

### 🇬🇷 Το Κίνητρο
Στην εποχή της παραγωγικής τεχνητής νοημοσύνης, τα codebases κατακλύζονται από "slop" — κώδικα χαμηλής προσπάθειας και γενικευμένης φύσης, που αν και λειτουργεί, στερείται "ψυχής" και βαθιάς πρόθεσης.
- **Η Παγίδα του "Παπατζή"**: Η AI λειτουργεί συχνά ως ένας ψηφιακός "παπατζής", παράγοντας κώδικα που φαίνεται επαγγελματικός στην επιφάνεια, αλλά στερείται των οργανικών ακανόνιστων στοιχείων και της βαθιάς λογικής της ανθρώπινης σκέψης.
- **Διατήρηση της Μαστοριάς**: Αυτό το εργαλείο προασπίζει την ποιότητα έναντι της ποσότητας. Είναι για προγραμματιστές που πιστεύουν ότι κάθε γραμμή κώδικα πρέπει να αντικατοπτρίζει φροντίδα και ανθρώπινη λογική.

## 🇺🇸 English Version

### 📦 The Ecosystem
Papatzis Spotter is more than just an app; it's a multi-layered defense system:

1.  🛡️ **Papatzis Orchestrator**: The central Command Center (GUI) to manage installation, CLI paths, and AI agent shielding.
2.  🖥️ **Papatzis UI (Desktop App)**: A premium Tauri-based diagnostic tool with real-time radar charts and deep AST inspection.
3.  💻 **Papatzis CLI**: A high-performance command-line tool for local and CI/CD slop detection.
4.  🤖 **PapatzoSkill (Global AI Shield)**: Injects strict anti-slop rules into **Cursor, Gemini CLI, and GitHub Copilot** to force AI agents to write better code.
5.  🪝 **Git Bouncer**: A pre-commit hook that automatically blocks any commit containing suspicious AI-generated slop.

### ✨ Key Features
- **5 Pillars of Analysis**: Naming, Structural, Comments, Redundancy, and AI Suspicion.
- **Deep Scan Mode**: Mathematical entropy and semantic drift analysis for "pro-papatzis" detection.
- **Humanity Shield**: Recognizes human markers (TODOs, hacks, slang) to reduce false positives.
- **Case File Export**: Generates detailed forensic Markdown reports for your code.
- **Privacy First**: 100% offline analysis. No code ever leaves your machine.

### 🚀 Getting Started
1.  **Run the Orchestrator**: Launch `launcher/main.py` (or the compiled EXE).
2.  **Launch UI**: Use the Orchestrator to open the Desktop App.
3.  **Shield Up**: Click "Shield AI Agents" to enforce rules globally.
4.  **Bouncer**: Click "Git Bouncer" to protect your current repository.

## 🇬🇷 Ελληνική Έκδοση

### 📦 Το Οικοσύστημα
Ο Papatzis Spotter είναι ένα σύστημα άμυνας πολλαπλών επιπέδων:

1.  🛡️ **Papatzis Orchestrator**: Το κεντρικό Κέντρο Ελέγχου (GUI) για τη διαχείριση της εγκατάστασης, του CLI και της θωράκισης των AI agents.
2.  🖥️ **Papatzis UI (Desktop App)**: Premium διαγνωστικό εργαλείο (Tauri) με radar charts και βαθιά AST ανάλυση.
3.  💻 **Papatzis CLI**: Εργαλείο γραμμής εντολών για γρήγορους ελέγχους και ενσωμάτωση σε CI/CD pipelines.
4.  🤖 **PapatzoSkill (Global AI Shield)**: Κάνει "ένεση" κανόνων σε **Cursor, Gemini CLI και GitHub Copilot**, αναγκάζοντας τα AI να γράφουν ποιοτικό κώδικα.
5.  🪝 **Git Bouncer**: Ένας "Πορτιέρης" (pre-commit hook) που μπλοκάρει αυτόματα commits που περιέχουν slop.

### ✨ Χαρακτηριστικά
- **5 Πυλώνες Ανάλυσης**: Ονοματολογία, Δομή, Σχολιασμός, Πλεονασμός και Υποψία AI.
- **Deep Scan Mode**: Μαθηματική ανάλυση εντροπίας και semantic drift για τον εντοπισμό "επαγγελματιών παπατζήδων".
- **Humanity Shield**: Αναγνώριση ανθρώπινων ιχνηλατών (TODO, FIXME) για μείωση των false positives.
- **Case File Export**: Εξαγωγή αναλυτικών φακέλων "λαδιάς" σε μορφή Markdown.
- **Privacy First**: 100% τοπική ανάλυση. Ο κώδικάς σας δεν φεύγει ποτέ από τον υπολογιστή.

### 🚀 Ξεκινώντας
1.  **Τρέξτε τον Orchestrator**: Ανοίξτε το `launcher/main.py` (ή το EXE).
2.  **Launch UI**: Χρησιμοποιήστε το κουμπί "Launch" για να ανοίξετε το Desktop App.
3.  **Shield Up**: Πατήστε "Shield AI Agents" για να επιβάλλετε τους κανόνες παγκόσμια.
4.  **Bouncer**: Πατήστε "Git Bouncer" για να προστατέψετε το τρέχον project σας.
